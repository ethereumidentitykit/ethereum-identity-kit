# Profile Card

Compact identity card for an Ethereum address, ENS name, or EFP list. Supports follower state, socials, stats, POAPs, and an optional follow button.

Also available as a **composable slot API** (`ProfileCard.Root`, `ProfileCard.Avatar`, …) — see the **CustomSlottedLayout** story.

[Full docs →](https://ethidentitykit.com/docs/profile-card)
